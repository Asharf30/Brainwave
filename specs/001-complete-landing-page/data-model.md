# Data Model

All data is static and should be placed in `src/constants.jsx` alongside any existing constants.

Entities:
- **Feature**: id, title, text, icon
- **Benefit**: id, title, text, backgroundUrl, iconUrl, imageUrl, light
- **Testimonial**: id, name, role, text, avatarUrl
- **FAQItem**: id, question, answer
- **FooterLink**: id, title, url, iconUrl
