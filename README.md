# Brainwave

Live Demo: [https://brainwave-two-ashraf.vercel.app](https://brainwave-two-ashraf.vercel.app)

## Project Structure

```text
BRAINWAVE/
├─ public/                       # Static public assets
├─ src/
│  ├─ assets/                    # Images, SVGs, and other media
│  ├─ Components/                # Reusable UI components and sections
│  │  ├─ design/                 # Background and design elements
│  │  ├─ Header.jsx              # Navigation bar
│  │  ├─ Hero.jsx                # Main hero section
│  │  └─ ...                     # Other sections (Benefits, Footer, etc.)
│  ├─ constants.jsx/             # Static data and configuration
│  ├─ index.css                  # Global Tailwind CSS file
│  ├─ App.jsx                    # Route map and main layout
│  └─ main.jsx                   # React bootstrap and DOM rendering
├─ index.html
├─ package.json
├─ vercel.json                   # Vercel SPA routing configuration
└─ vite.config.js
```

## Tools Built With

- **React** (UI Library)
- **Vite** (Build Tool)
- **Tailwind CSS** (Styling)
- **GSAP** (Animations)
- **React Router** (Routing)
- **React Just Parallax** (Parallax effects)
